Face Recognition

Click on the image, register an account, and check out my Face Recognition app! This is an app that takes a submitted image URL, sends it through the Clarifai API, and generates a box around the detected face in the image. I built this project from the ground up with HTML, CSS, JS, Node.js, Express, and PostgreSQL. It is currently hosted on Heroku (my free personal account) so stick with it if it doesn't load as fast as it could!

PokeFinder

Click on the image to take a look at my simple react app, Pokefinder! This app dynamically updates state based on the user's input. That state is then used to communicate with the free PokeAPI to generate a series of cards that match the input. Created with React.js and hosted through GitHub pages.

Background Generator

Click the image, and generate your own custom background gradient between two colors of your choice. This site is made using standard HTML, CSS, and JS. After selecting the two colors you would like to generate a gradient between, the site generates the necessary CSS for you to copy and paste into your own program. My Background Generator is currently hosted on GitHub pages.
